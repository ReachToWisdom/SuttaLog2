# TaskExecution — 테스트/검증 계획

## 빌드 검증

```bash
npm run build     # tsc 타입체크 + vite 프로덕션 빌드
npm run dev       # 개발서버 :3023 확인
```

- [ ] `npm run build` 에러 없이 통과
- [ ] dist/ 폴더에 해시 포함된 파일명 생성 확인
- [ ] 개발서버에서 전체 화면 탐색 가능

## 화면별 검증

### 홈 (`/`)
- [ ] 시간대별 인사 메시지 표시
- [ ] 연속 학습일수 표시 (불꽃 뱃지)
- [ ] "이어서 학습하기" / "학습 시작하기" CTA 동작
- [ ] 오늘의 게송 표시

### 과목 (`/courses`)
- [ ] 5개 카테고리 접이식 동작
- [ ] **잠금 없이** 아무 과목이나 클릭 진입 가능
- [ ] 검색으로 과목 필터링
- [ ] 진행률 원형 표시 정확

### 학습 (`/learn/:id`)
- [ ] 각 스텝 타입 10종 정상 렌더링
- [ ] **목차 버튼** 탭 → 스텝 목록 오버레이 표시
- [ ] 목차에서 아무 스텝 탭 → 해당 스텝으로 이동
- [ ] **건너뛰기** 버튼 모든 스텝에서 표시 (마지막 제외)
- [ ] 하트 0 → 게임오버 화면
- [ ] 마지막 스텝 완료 → 완료 화면 (통계)
- [ ] 진도 localStorage 저장 확인
- [ ] 뒤로가기(←) 버튼 동작

### 복습 (`/review`)
- [ ] 완료 과 없으면 안내 메시지
- [ ] 완료 과 있으면 랜덤 복습 버튼 + 목록

### 설정 (`/profile`)
- [ ] 쓰기 모드 토글 → 학습에서 반영
- [ ] 소리 토글 동작
- [ ] 진도 초기화 → confirm → 초기화

## 캐시 검증

- [ ] dist/index.html에 no-cache 메타태그 존재
- [ ] dist/assets/ 파일명에 해시 포함 (`-[hash].js`)
- [ ] 코드 수정 후 재빌드 → 해시 값 변경 확인

## 동기화 검증 (Firebase)

- [ ] Google 로그인 성공
- [ ] 업로드 → Firestore에 데이터 저장
- [ ] 다운로드 → localStorage에 데이터 복원
- [ ] 로그아웃 동작

## 배포 검증

- [ ] `git push main` → GitHub Actions 빌드 성공
- [ ] Pages URL 접속 → 최신 버전 확인
- [ ] 캐시 문제 없이 최신 JS/CSS 로드
